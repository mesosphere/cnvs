Fluid-Width Containers

Use .container-fluid for a fluid container. A fluid container is one where the width of the container is fluid relative to the width of it's parent. By default, a fluid container has outside margins on the left and right. The width of these margins can be modified with provided classes or removed entirely. For small devices, a fixed and a fluid container behave the same. 

<div class="container-fluid">
  <!-- Container content goes here -->
</div>

Fluid-Width Container Options

By default, margins are applied evenly to both the left and right of the fluid container. You can easily adjust this behavior by adding a size modifier to the base .container-fluid class. For example .container-fluid-narrow will decrease the margins and .container-fluid-wide will increase the margin. Furthermore, if you are seeking to change the margin in only a single direction, you can add an additional direction modifier. For example, .container-fluid-wide-left increase the margin only the left of the fluid container.

Size Modifiers

To adjust margin apply one of the available container-specific size classes. For example, simply using the class .container-fluid-narrow will reduce margins evenly on each side of the fluid container.

Class Description
.container-fluid-flush Outside margins are removed entirely.
.container-fluid-narrow Outside margins by default are decreased by 50%
.container-fluid-narrower Outside margins by default are decreased by 25%
.container-fluid-wide Outside margins by default are increased to 200%
.container-fluid-wider Outside margins by default are increased to 400% 
<div class="container-fluid container-fluid-narrow">
  <!-- Container content goes here -->
</div>

Size Direction Modifiers

Add -left or -right to the end of the size modifier class to adjust margin only in the direction implied by the direction modifier.

Class Description
.container-fluid-*-left Adjust margin to the left of the container.
.container-fluid-*-right Adjust margin to the right of the container. 
<div class="container-fluid container-fluid-wider-left container-fluid-narrow-right">
  <!-- Container content goes here -->
</div>
